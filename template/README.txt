- checkin.bat
Позволяет с помощью использования данной консольной комманды прочитать данные файла "in.txt". В параметрах при запуске ничего не указывается.


- cpl.bat
Позволяет с помощью использования данной консольной комманды выполнить файл с расширением ".cpp". Пример записи комманды в консоли: cpl my_program.cpp


- updin.bat
Позволяет с помощью использования данной консольной команды обновить данные файла прямо из консоли. !Данные вставляются из буфера обмена!
+ имеет параметр -l
При использовании параметра, также запускает нужный вам файл с расширением ".cpp". Пример записи комманды в консоли: updin -l my_program.cpp


- updin.py
Используется для работы updin.bat
